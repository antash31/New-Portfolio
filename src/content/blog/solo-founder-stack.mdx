# The Solo Founder Stack — 2026 Edition

Running a company alone doesn't mean running it small. Here's the exact stack I use.

## Principles

- **Default to async.** Meetings are a tax on momentum.
- **Automate before you delegate.** Cheaper, faster, more reliable.
- **One source of truth.** If it lives in two places, it lives in zero.

## The stack

- **AI layer:** Custom GPT pipelines, Claude for deep research, Cursor for code
- **Ops:** Linear for tasks, Notion for docs, Cal.com for scheduling
- **Content:** Next.js + MDX for the site, Descript for video, Buffer for distribution
- **Finance:** Mercury + Stripe. That's it.

Simplicity scales. Complexity collapses.
